Some of Our Products

Chainflux has built in-house products on private Blockchain networks to solve business problems that benefit from blockchain's immutability, shared ledger, and traceability.
  • Chainflux Trace

    Coordination between different players in a supply chain for tracking digital and physical assets is a problem due to data existing in silos and lack of inter-operability betweeen ERP systems. Chainflux trace aims to solve this problem by enabling a Blockchain based solution to track and manage assets across ERP systems as well as be a stand-alone solution for any member of the supply chain.

  • Chainflux-BAAS

    Chainflux-Blockchain As A Service is a platform that enables any organisation to easily start up a private Blockchain network with a defined set of characteristics, defined number of participants in the system, defined access rights, and addition of certain smart contracts such as escrows or voting mechanisms

  • Chainflux Settled

    Chainflux settled is a financial settlement application that enables any financial institution/NBFC to implement it's private Blockchain network to settle any digital assets or financial derivatives. The smart contracts created enable instant settlement, and transparency into which contracts are settled and between which parties they were settled.
